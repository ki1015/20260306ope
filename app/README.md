# オペ戦 理論クイズ アプリ

通勤・通学中にサクッと学び直せる問題集アプリです。

**みんなに共有する方法:** プロジェクトフォルダ直下の **`共有方法.md`** を参照してください（Netlify・GitHub Pages・Vercel などで公開できます）。

**流れ:** 問題 → 回答（4択 or ○×）→ 正誤表示 → 解説 → 次へ → … → 結果

- **結果の蓄積**: 解答ごとに正誤を記録（ブラウザのローカルストレージ）。累計正答率・得意・苦手がわかります。
- **間違えた問題だけ復習**: 開始画面・結果画面から「間違えた問題だけ」で苦手を集中的にやり直せます。
- **得意・苦手の表示**: 結果画面でカテゴリ別の正答率を表示し、苦手なカテゴリを把握できます。

## ローカルで起動する

**いちばん簡単:** プロジェクトのフォルダで **`start-app.bat`** をダブルクリック（Windows）。  
サーバーが起動し、開くURLと携帯用の案内が表示されます。

**手動で起動する場合:**

1. プロジェクトのルート（`20260305オペ戦` フォルダ）でターミナルを開く
2. 次のどちらかを実行:
   ```bash
   npx serve . -p 3000
   ```
   または（Python がある場合）
   ```bash
   python -m http.server 3000
   ```
3. PCのブラウザで **http://localhost:3000/app/** を開く

## 携帯で同じ画面を見る方法

PCと携帯が **同じWi-Fi** につながっている必要があります。

1. **PCのIPアドレスを調べる**
   - Windows: ターミナルで `ipconfig` を実行 → 「IPv4 アドレス」の数字（例: 192.168.1.5）をメモ
   - `start-app.bat` で起動した場合は、表示された「このPCのIP」をメモ

2. **携帯のブラウザで開く**
   - アドレス欄に **`http://[PCのIP]:3000/app/`** と入力  
     例: `http://192.168.1.5:3000/app/`
   - Python で起動した場合も同じ（ポート 3000 なら同じURL）

3. **アプリのように使う（任意）**
   - iPhone: Safari で開く → 共有 → 「ホーム画面に追加」
   - Android: Chrome で開く → メニュー → 「アプリをインストール」または「ホーム画面に追加」

※ データ（正誤の記録）は **各端末のブラウザ** に保存されます。PCと携帯では別々の記録になります。

## パスワードロック（公開時）

アプリを開くと最初にパスワード入力画面が表示されます。正しいパスワードを入力すると利用できます。

- **初期パスワード:** `takoyaki`
- **変更する場合:** `app/js/app.js` の先頭付近にある `LOCK_PASSWORD = 'takoyaki'` の `'takoyaki'` を任意の文字列に書き換えてください。
- **解除の有効期限:** ブラウザのタブを閉じるまで有効です。再度開いたときはパスワードの再入力が必要です。
